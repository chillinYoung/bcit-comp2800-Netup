# COMP2800 Project
- BCIT course for students to work in teams to build web applications

## Start Guide

1. Ensure you have firebase-tools installed as your cli

```
npm install -g firebase-tools
```

2. Go into the functions folder and install the npm packages

```
cd functions
npm install
```

3. If you are not already logged into your firebase account, you need to first login

```
firebase login
```

4. Pick a project to run this app

```
firebase use --add
```

5. Type in the following command to serve up the app locally

```
firebase serve --only hosting,functions
```

6. Open your browser and type: http://localhost:5000
